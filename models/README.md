# Models

This directory contains references to models used by ProjectClipsAI.

**Model files are stored externally — never inside this repository.**

See `docs/MODEL_LIBRARY_POLICY.md` for the full policy.

## External Model Library

Configure your model library path in Stability Matrix to point to an external location, e.g.:

```
D:\AI_Models\
├── checkpoints\
├── loras\
├── vae\
├── video\
├── upscale\
└── clip\
```

## Registry

All models are cataloged in `registry/MODEL_REGISTRY.md`. Check there before downloading any model.
