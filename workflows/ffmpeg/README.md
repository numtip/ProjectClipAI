# FFmpeg Scripts

Store FFmpeg scripts for video assembly and processing.

## Convention

- File name: `NNN-description.ps1` or `NNN-description.bat`
- Comment the top of each script with: purpose, inputs, outputs, usage

## Typical Operations

- Concatenate multiple clips
- Add crossfade transitions
- Overlay text or images
- Sync and merge audio
- Export to MP4 with optimal settings

## Usage

All scripts should be run from the repository root or accept paths as parameters.
