# Export QA

**Purpose:** Quality assurance for final MP4 exports before delivery.

## Pre-Export Checklist

- [ ] All shots rendered and reviewed (per RENDER_QA)
- [ ] FFmpeg assembly script is correct
- [ ] Shot order matches storyboard
- [ ] Transitions are appropriate (if any)
- [ ] Audio tracks are synced (if any)

## Export Verification

- [ ] Final MP4 plays correctly
- [ ] Total duration matches project brief
- [ ] Resolution and bitrate are acceptable
- [ ] File exports to project exports folder
- [ ] File size is reasonable for the content

## Archive

- [ ] render-notes.md is complete
- [ ] registry/PROJECT_REGISTRY.md updated
- [ ] Final clip path documented in registry
- [ ] Source files preserved in project folder
