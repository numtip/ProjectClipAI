# Render QA

**Purpose:** Quality assurance checklist for rendered video clips.

## Visual Quality

- [ ] No visible artifacts (blockiness, color banding, tearing)
- [ ] Motion is smooth (no stuttering or frame drops)
- [ ] No flickering between frames
- [ ] Subject remains consistent throughout clip
- [ ] Background is stable (no warping unless intended)
- [ ] Colors are accurate and pleasing

## Technical Quality

- [ ] Resolution meets project requirements
- [ ] Frame rate is consistent
- [ ] Duration matches expected length (±0.5s)
- [ ] No audio sync issues (if audio is included)
- [ ] File plays in standard media player (VLC, MPC-HC)

## Render Issues to Flag

- Visible seams or cuts in video
- Subject distortion or morphing
- Background flicker or noise
- Color shift mid-clip
- Premature cut-off
- Corrupted output file
