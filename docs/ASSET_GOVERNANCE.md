# Asset Governance

> Standards for managing shared reference assets in ProjectClipsAI.

---

## Asset Structure

```
assets/
├── references/       # Reference images, mood boards, style guides
├── images/           # Generated key images, source images
├── audio/            # Sound effects, music, voiceovers
└── fonts/            # Typeface files for titles and overlays
```

---

## Rules

### 1. Source Tracking Required

Every asset must have a source recorded in `registry/ASSET_REGISTRY.md`:
- If downloaded: URL and license
- If generated: prompt, workflow, model, seed
- If created externally: author or tool

### 2. Generated Assets Recorded

Assets generated by ComfyUI for a specific project belong in the project's `assets/` folder, not the shared `assets/`. Update `registry/ASSET_REGISTRY.md` only when an asset becomes shared across projects.

### 3. Final Assets Registered

Assets used in a final export must be registered in `registry/ASSET_REGISTRY.md` with their project reference.

### 4. No Duplicate Assets

Check `registry/ASSET_REGISTRY.md` before adding any asset. If the asset already exists, reference it rather than re-copying.

---

## Naming Convention

`{category}-{description}-{resolution-if-image}.{ext}`

Examples:

```
reference-cyberpunk-moodboard-1920x1080.png
image-warrior-red-cloak-001-1024x1024.png
audio-ambient-forest-loop-30s.mp3
font-inter-variable.ttf
```

### Rules

1. Use lowercase kebab-case.
2. Include resolution for images.
3. Include duration for audio.
4. No spaces.
5. No "final", "new", or "working" qualifiers.

---

## Category-specific Rules

### References

- Keep lightweight (compressed JPEG or PNG).
- Include attribution or URL in `ASSET_REGISTRY.md`.
- Max 1920px on longest edge.

### Images

- Retain original resolution from generation.
- Do not recompress generated images (loss of detail for video).
- Name includes the seed or attempt number.

### Audio

- Prefer MP3 or AAC for compatibility.
- Document duration and sample rate in registry.
- Verify no licensing restrictions.

### Fonts

- Store only open-license fonts.
- Document font family, weight, and license in registry.
- No system fonts (assume available on target system).
