## Status Legend

- **Active** — Currently used and tested
- **WIP** — Work in progress, not yet stable
- **Archived** — Replaced or obsolete

## Policy

1. Start with one working workflow. Expand only when stable.
2. Store workflow JSON exports in `workflows/comfyui/`
3. Document all required custom nodes for each workflow
4. Do not install custom nodes without documenting them here first
